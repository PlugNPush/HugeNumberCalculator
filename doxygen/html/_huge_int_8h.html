◆ createHugeIntFromHugeInt()

HugeInt* createHugeIntFromHugeInt ( const HugeInt *  hugeInt,
const unsigned int  size 
)

Function to create a HugeInt initialized with the size initial values of another HugeInt. The sign is always copied.
Example :

// Copy the sign and all the values of hugeIntSource in the new hugeIntDest
HugeInt* hugeIntDest = createHugeIntFromHugeInt(hugeIntSource, -1);
// Copy the sign and just the first 5 values of hugeIntSource in the new hugeIntDest
HugeInt* hugeIntDest = createHugeIntFromHugeInt(hugeIntSource, 5);
Parameters
hugeIntthe HugeInt to copy.
sizethe number of values to copy, enter -1 to copy all the values.
Returns
the HugeInt's pointer, if the HugeInt is created.
NULL, if the HugeInt could not be created.
