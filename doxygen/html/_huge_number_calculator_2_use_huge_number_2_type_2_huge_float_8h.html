HugeFloat* createHugeFloatFromHugeFloat ( const HugeFloat *  hugeFloat,
const unsigned int  size 
)

Function to create a HugeFloat initialized with the size initial values of another HugeFloat. The sign and the exponent are always copied.
Example :

// Copy the sign and all the values of hugeIntSource in the new hugeIntDest
HugeFloat* hugeFloatDest = createHugeFloatFromHugeFloat(hugeFloatSource, -1);
// Copy the sign and just the first 5 values of hugeIntSource in the new hugeIntDest
HugeFloat* hugeFloatDest = createHugeFloatFromHugeFloat(hugeFloatSource, 5);
Parameters
hugeFloatthe HugeFloat to copy.
sizethe number of values to copy, enter -1 to copy all the values.
Returns
the HugeFloat's pointer, if the HugeFloat is created.
NULL, if the HugeFloat could not be created.
